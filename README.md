 SkyCast Pro – Weather Forecast App

A modern, responsive weather web application built using HTML, CSS, and JavaScript with real-time weather data integration using the OpenWeather API.

📌 Live Features
🔍 Search weather by city name
🌡 Real-time temperature display (°C)
☁ Current weather condition (clear, cloudy, rain, etc.)
💧 Humidity tracking
📅 4-day weather forecast
👁 Visibility and wind speed data
📊 Interactive forecast cards
🖼 Weather icons from API
📱 Fully responsive design (mobile, tablet, desktop)
🎨 Modern glassmorphism UI with animated background
🪟 Detailed weather modal popup

🧠 Project Highlights
Built using vanilla JavaScript (no frameworks)
API integration with OpenWeather
DOM manipulation for dynamic UI updates
Local storage used to remember last searched city
Responsive grid-based layout
Clean UI with animated gradient background
🛠 Tech Stack
HTML5
CSS3 (Glassmorphism + Responsive Design)
JavaScript (ES6+)
OpenWeather API
